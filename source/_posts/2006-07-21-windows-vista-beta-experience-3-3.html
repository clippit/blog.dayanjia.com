终于把Vista5472安装完了，心中真是一阵狂喜！
待心情渐渐平静，我决定开始写一篇连载，题目叫做《一个爱好者眼里的 Windows Vista》。
之所以要强调“爱好者”这三个字，
是因为我不像盆盆、刘晖那样的微软MVP从技术的层面大谈Vista的新功能、新特性，
我只是从“平民立场”出发，
由Vista说功能，由功能说变化，由变化说体验，由体验说感受。
我希望能将一个全景的Vista展示给大家看；
能让大家感觉到Vista的方方面面细细微微；
能让我们更加充分地了解这个即将或者已经走进我们生活的事物。
我同时也希望大家能够喜欢我的文章！
好了，这就是我的序言，一个貌似序言又不像序言的序言。
今天是这篇连载的第一集，我们从Vista的安装开始说起——

注意：本文章所有图片均可点击放大，多图杀猫！

首先简要地说一下我的系统配置：

  • CPU：Intel Celeron D 346 (3.06GHz)
  • 主板：Intel D945PLRN (i945PL芯片组)
  • 内存：Kingston DDR2 533 512M ×2
  • 硬盘：希捷酷鱼 7200.9/80G/8M/SATAII
  • 显卡：nVIDIA GeForce 7300 GT  (256 MB)

下面就要开始安装Vista了。首先当然需要Vista5472的光盘镜像，有需要的朋友可以到http://www.vistafans.com/thread-31283-1-1.html去下载。接着还需要准备一个虚拟光驱软件，我使用的是DAEMON Tools，小巧、方便。用它加载下载好的ISO，如果你打开了自动播放，安装界面就会出来了；没有打开也没有关系，进入光盘目录双击“Setup.exe”文件就可以啦！反正不管你用哪种方法，只要看到了下面这样的窗口就算开了一个好头。

俗话说，好的开头是成功的一半，现在应该感到有一点兴奋了吧！因为你已经看到了Vista的招牌式的极光画面和颇有质感的按钮。好了，欣赏完毕以后点击“Install now”吧！接着安装程序会做一些不为人所知的事情，只会在全屏的画面下告诉你“Please wait”，不知道它在搞什么飞机。

大家可以继续欣赏激光背景等你欣赏够了基本上它也完成工作了，接下来会进入正式安装的第一部分——collecting information（收集信息），第一步它会询问你是否要到网上获取最新的更新，按它推荐的选吧！点击第一个选项。在这个窗口下面有一个“I want to help make Windows installation better”选项，如果你怕微软把你的小秘密泄露出去的话就把它取消我刚才说，这已经开始了正式安装了，前面的都是前奏，因此有必要再兴奋一下~~~

如果你在上一步选择了获取更新的话，你就会看到如下的界面，这时候让我们再欣赏一下极光吧

接下来微软会向你索取一样东西，这个东西可以说也是盗版的关键物品，通常在盗版光盘中这个东西会包含在一个名为sn.txt的文件中。你知道这是什么东西吗？不知道的话请看下图：

如果你在Beta2公测的时候已经申请过序列号了，那么就把它找出来复制进去吧！如果没有的话，可以用这个：2PHXF-9PBDW-D3WWY-CPDKD-XG87V，不过这个序列号是无法激活的~~~

按下“Next”，会出现授权协议，废话当然接受（霸王条款害死人呐……）

再“Next”，下一步可以选择是升级安装还是全新安装。大家可以看到我的升级安装是灰色的，下面有详细的说明，说一是我C盘的可用空间不足，二是现有操作系统的语言不符。不管怎么说，反正是不能升级，不过我也不推荐大家升级，毕竟Vista还没有正式推出，这么早就把原来好端端的XP推掉实在有一点不忍心啊！

点击“Custom”以后，便进入了选择安装盘的阶段，这里列出了可选的安装盘。前面已经说道，我使用的是SATAII的硬盘，在这里并没有出现什么识别不出的问题。不过有人反映他的硬盘识别不出来，我估计使用了Raid之类的恐怕会出现这种问题，这时候点击“Load Drive”加载好驱动以后应该就没有什么问题了。

选择好以后点击“Next”。在这里我的电脑还出现了一个小插曲，弹出来一个警告，当时还真把我吓了一下，以为无法安装呢！我估计这是因为我把XP里的“我的文档”移动到D盘去了，而且有三个软件安装在D:\Program Files文件夹下。我真不敢轻举妄动，干脆来了一个格式化大法：先把D盘里的东西搬到E盘里去，然后再格式化D盘！呵呵~~够狠吧？这里顺便说一下，Vista只能安装到NTFS分区里面去，而且这个分区最好要大于15G，更好的是里面不要有文件

接下来你所要干的事情就只有两个字了：等待。不过也不要担心，不会等待多长时间的。中央的窗口中显示着安装过程，同时下面的进度条也开始了第二部分：Installing Windows，也就是说，现在开始正正式式地安装Vista了！要不要再兴奋一下？呵呵~~可以看到，第一项任务是复制文件到你的硬盘里面去，让我们耐心地等待……

在我的机子上，这一阶段花费了大约4分40秒的时间。接下来是第二项工作——展开文件。

      

相信你很早就注意到，安装的窗口似乎使用了Aero主题吧！其实如果你试图移动这个窗口，你会发现：这哪里是什么窗口！根本就是“焊”在背景上不能移动的！其实这个窗口也是背景的一部分啊！哈哈，你想在XP下就用Vista的专利？下辈子再想吧！不扯了，在展开文件达到27%进度的时候，安装程序要重新启动了，这意味了我的截图软件SnagIt8马上就要完成它的使命功成身退了，这也意味着接下来的安装过程没有截图了……没办法，谁叫我没有在虚拟机里面安装呢？好了，让我们看看SnagIt的“遗照”吧！

重新启动以后，可以看到启动时的滚动条了，但是奇怪的是只有一个白色的滚动条和下面的“©2006 Microsoft”之类的字以外没有其它东西，真是比较诡异！滚动条转了10圈以后，终于又进入了安装程序，再经过一段时间的等待，又开始继续展开文件的工作了。整个展开文件的过程（不包括重新启动，只包含屏幕上显示“Expanding files”的时间）共耗时约8分40秒。

接下来是“Installing features”安装特性，几乎是一闪而过，然后是“Installing updates”安装更新，耗时小于10秒。然后又是重新启动，这一回我看到了Vista的新版的Boot Manager启动管理器，不过只是一闪而过。接下来滚动条转了8圈，进入后屏幕上显示出一行字：Please wait a moment while windows prepares to start for the first time.......后面跟了一长串了省略号。1分钟后Windows终于准备好第一次启动了。你可别以为安装已经完成了！进入系统以后，我等了约6分38秒的时间系统没有任何提示，只看到硬盘灯在闪，CPU风扇在转。我估计这个时间Vista在给你安装各种驱动程序，因为这期间我的屏幕黑了两下，这是明显的安装显卡驱动的情形啊！接下来是最后一个部分——Completing installation完成安装，等待了约1分钟后终于完成了！系统再一次重新启动，这一回滚动条只转了2圈半。

进入Vista系统了！真是功夫不负有心人！！你可不要太激动，还有一些后续的扫尾工作需要完成呢！首先需要设置登陆用户名和密码，然后是设置计算机名和桌面壁纸，接下来是设置自动更新，选择“Use recommended settings”使用推荐设置得了。接下来，系统会提示你“You're ready to start”，这时候，单击上面的Windows徽标（如果你愿意单击下面的“Start”按钮也可以，但是总觉得少了一分隆重感）。这时候，你还没有真正进入Vista 的世界呢！真够烦的~~~屏幕下方会显示一个长长的进度条，提示你“Please wait while windows checks your computer performance”，同时上方简洁地介绍着Vista的各个组件，每一个组件似乎都有个“口号”。检查计算机性能在我机器上消耗了大约2分45秒的时间。随后，你就真正地进入了Vista的世界！恭喜你！！

纵观整个安装过程，复制文件4分40秒，展开文件8分40秒，安装更新<10秒，准备第一次启动1分钟，安装驱动6分38秒，完成安装1分钟，检查性能2分45秒，一共耗时约24分45秒，再加上各种设置、重新启动和一些乱七八糟的等待，从开始安装到可以开始使用Vista的时间在30分钟左右。可以说这个时间是很快的。据说这得力于一种类似于Ghost还原硬盘的技术，把一个镜像直接放到硬盘里面去了。看来从开始安装，Vista就使用了各种新技术啊！微软在它上面下的功夫还真不少！（虽然他们都是为了＄＄＄＄＄）

一个爱好者眼里的 Windows Vista ⑴ 到这里就结束了，不知你看得过瘾不过瘾？从下一集开始，我就要正式进行Vista的体验了，欲知详情，请看下集！谢谢……
