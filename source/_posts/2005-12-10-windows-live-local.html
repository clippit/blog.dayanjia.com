正如标题中做说的，Windows Live Local 正式出炉啦！！
地址：http://local.live.com/

 

没错，Windows Live Local 的前身就是炒得沸沸扬扬的 MSN Virtual Earth，想当初这个服务刚一推出，便引来了无数骂声，因为它的使用并不像宣传中的那么好，有时甚至无法显示图片，据说图像数据的年份也是很久以前的，因为有很多地方居然是黑白的图片！还有，宣传中出现的45度鸟瞰图在 MSN Virtual Earth 里根本不见踪影！！

不过微软似乎并没有把它当回事。十一月初，Windows Live 的名字打响了！Windows Live （http://www.live.com/）是微软推出的一项在线应用服务集合。这项服务通过网络浏览器接口提供诸如Microsoft Office之类的服务。2005年11月1日星期二产品进入β版。

于是，Windows Live Local 成了 Windows Live 的一部分……

“重命名”之后，Windows Live Local 的主要内容并没有多少变化，黑白的图片还是黑白的，模糊的图片还是模糊的——可以说，在这一点上，Windows Live Local 完败于 Google 的 Google Local（即原先的Google Maps）和 Google Earth，因为在它们中根本看不到黑白的图片，而且我可以打包票高清晰的图片绝对比 Windows Live Local 里的多！

当然，Windows Live Local 也有它优势的一面。比如说它提供的地名信息远远比Google的要全得多，在我家乡附近的一些乡镇的名字也是非常全面的，而Google里连“Nantong”这样的城市都没有！

最令人兴奋的当属“bird's eye”了！也就是说，令人心动的鸟瞰功能终于浮出水面！！不过，这只是在美国的一些城市中才会有的（郁闷ing……）。它的清晰程度没话说，从比例尺上可以看出，最清晰的图片是“20yds”，也就是20码，合约18.3米——这是什么概念？如果你家住在5楼或者6楼的话，你从阳台上看到的窗外的景色都可以在 Windows Live Local 一览无余！！下面这张图片就是在 Windows Live Local 中的自由女神像，如果去掉周围的一些控制面板和横幅导航条的话，相信每一个人都会认为这是一张你拍摄的自由女神像的照片！！！
