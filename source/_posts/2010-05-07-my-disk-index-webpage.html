自从买了那块1TB的移动硬盘后，我就开始了疯狂的填塞活动，如今移动硬盘的剩余容量只有五六十GB了。除了一些自己备份的资料以外，移动硬盘内存有大量多媒体资源，简单地分类下，包括高清电影、电视剧集、大型软件、电脑游戏和学习资料。同时由于仙林校区学生宿舍残酷得令人发指的网络环境和上网资费，许多同学纷纷在我这里寻找自己想要的资源，数十GB的东西，一会儿就能拖到自己的电脑里，不仅省下了宿舍的上网时间，节约了资金，最最关键的还是避免了自己开迅雷时舍友的怒目圆瞪：“你小子又开迅雷，哥打DotA都卡啊！！！”所以说，我这块移动硬盘为同学宿舍关系的和谐繁荣发展做出了不可泯灭的卓越贡献！不过随着业务越来越繁忙，我越来越觉得有必要给它建立一个索引的了，所以就有了这个小网站，这样大家就可以清楚地了解这块移动硬盘中的公共多媒体资源。

围观地址：http://mydisk.dayanjia.com

做这个网页，当然也是为了实战性地学习一些东西，做这个网页学习并用到了以下技术：

  • SQLite3，一个单文件的轻量级关系型数据库
  • PHP Data Objects (PDO)，是PHP提供的数据库访问抽象层
  • Web 2.0 Mashup，电影和电视剧集资源中，可以通过豆瓣网获取资源详细介绍、评分等信息
  • XPath选择器，主要用于分析豆瓣API返回的Atom XML
  • jQuery 页面动画效果
  • jQuery Ajax，资源的详细信息通过Ajax后台获取

整个页面采用了灰色的色调，CSS里都是#999, #DDD这样的，不过加上了动画效果以后，看上去还是不错的。不过IE6这样的古董浏览器就悲剧了，因为IE6只支持<a>标签的:hover伪类，而网页中没有用<a>标签来展现条目。不使用<a>标签也就意味着这个网页的JavaScript并非渐进增强，在一些简陋的浏览器中就只能看到五个大标题，不能向下展开了。不过经过实际测试，iPod Touch中的Safari还是可以正常浏览该网页的，看来移动版的Safari果然很强大。
