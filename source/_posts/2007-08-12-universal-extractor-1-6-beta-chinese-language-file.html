强大的解包软件又出了新版本，不过现在还是beta版，不过已经增加了许多功能了！原来附带的1.5版简体中文语言文件有点力不从心了，所以我就弄了1.6beta的语言文件，让它更加完美。

附这次beta的更新列表：

1.6 beta (08/09/2007):

  添加 支持个人的用户设置，为 Vista 提供更好的支持；
        默认情况下，独立版本启用，安装版本禁用
        当启用时，像先前版本一样，UniExtract 使用单独的 .ini 文件
        当禁用时，个人的设置和历史保存在注册表 HKCU 根键下
        这可以通过修改设置文件中的 globalprefs 选项来更改
  添加 菜单栏到主窗口GUI；
        包含了退出、设置、访问UniExtract网站的选项
  添加 分离设置对话框，提供更加方便访问所有选项的方法；
        可以通过编辑菜单调用或通过“/prefs”参数
  添加 FEAD 优化的软件包的支持（例如 Adobe Reader 安装程序 ）
  添加 LZMA 压缩文件的支持
  添加 Nero NRG CD-ROM 映像（仅数据）的支持，通过 nrg2iso 程序
  添加 Reflexive Arcade 安装程序的支持，通过 RAIU 程序
  添加 WIM (Windows 映像格式) 映像的支持，通过 7-Zip 程序
  添加 “不是一个 InstallShield 安装程序”选项到 InstallShield 选择提取方式
        的对话框中来强制 UniExtract 处理 TrID 的错误
  添加 安装程序的组件区段；可以选择是否安装
        文档、语言文件和某些较大的程序
  添加 “发送到”选项到安装程序
  添加 解压 ASPack 和 UPX 文件的国际化支持
  添加 匈牙利语、葡萄牙语、罗马尼亚语、土耳其语和和巴伦西亚语(加泰罗尼亚)
        翻译文件
  添加 返回代码指出提取的状态（事实上在1.5版中已添加）：
        0 = 成功退出或用户干预取消
        1 = 理论上受支持的文件，但是提取失败
        2 = 调试文件无法写入，终止
        3 = 未知可执行文件 - 不能被提取
        4 = 未知的文件类型 - 不能被提取
        5 = 指定的输出目录无效
  修复 显示在 Inno Setup 提取过程中的调试信息
  修复 Microsoft 修复程序的支持（再次）
  修复 相关路径的支持
  修复 UNC 路径的支持
  修复 安装过程中缺失的西班牙语选项
  移除 对 Windows 9x 的支持，因为 AutoIt 新的 Unicode 版本；
          9x 不再有官方支持，甚至将不能运行
  移除 Adobe 具体报告（现归入一般的 FEAD 支持）
  更新 UniExtract 在提取可执行文件前提示用户；
          可以在警告选项中禁用
  更新 UniExtract 使 ACE, KGB, Pea, 和 StuffIt 支持变成可选
  更新 UniExtract 从安装根目录中读取 English.ini
  更新 UniExtract 默认输出调试文件到 %temp%
  更新 UniExtract 校验调试文件存放位置是否可以写入；
          如果所选文件夹测试失败则使用用户的临时文件夹
  更新 UniExtract 默认禁用添加扩展名选项
  更新 UniExtract 的更新日志，添加在1.5版中遗漏的的越南语翻译文件说明
  更新 TrID 检测 MS 自解压 CAB (类型 1) 档案文件
  更新 TrID 检测 Windows Installer (MSI) 软件包
  更新 TrID 检测 Zip 自解压档案文件
  更新 CD-ROM 映像支持，绕过 TrID 检测而仅根据扩展名
  更新 Windows Install 补丁 (.msp) 包括到 7-zip 选项
  更新 安装程序支持不在开始菜单中添加快捷方式
  更新 安装程序语言初始化代码以增强简明性
  更新 安装程序，需要管理员权限；
          非管理员用户只能使用绿色版本，
          或使用系统管理员安装好了后复制过来的版本
  更新 安装程序来防止在 Vista 下和 CHM 文件创建关联
  更新 安装程序支持新的 /nowarnexecute 参数
  更新 安装程序支持相反的添加扩展名的默认设置；
          现在使用 /appendext 启用来代替 /noappendext 禁用
  更新 安装程序添加 {app} in addition to {app}\bin to %PATH% if enabled;
          恢复了能够快速从命令行调用 UniExtract.exe 的能力
  更新 7-Zip 到 4.52 beta
  更新 innounp 到 0.19
  更新 Inno Setup 到 5.1.13
  更新 Pea 到 1.6 (不能使用更新的版本由于有问题的GUI控制)
  更新 UnRAR 到 3.70
  更新 UPX 到 3.01
